Jenis Data dan Pengelolaan Data dari Internet

Pada pertemuan pertama ini, peserta didik akan mempelajari berbagai jenis data serta cara mengakses dan mengelola data dari internet secara aman dan bertanggung jawab.

🧩 Tujuan Pembelajaran

✔ Peserta didik mampu membedakan jenis-jenis data.
✔ Peserta didik mampu mengakses dan mengelola data yang bersumber dari internet.
🌟 Pemahaman Bermakna

Peserta didik mampu membedakan jenis-jenis data serta mengakses dan mengelola data yang bersumber dari internet secara tepat, aman, dan bertanggung jawab.

1️⃣ Jenis-Jenis Data

A. Berdasarkan Bentuknya

📝 Data Teks
Data berupa tulisan seperti artikel, buku digital, chat, atau dokumen Word.
Contoh: Teks berita di website sekolah.

🔢 Data Angka (Numerik)
Data berupa angka yang dapat dihitung dan dianalisis.
Contoh: Nilai ujian siswa dalam tabel Excel.

🖼️ Data Gambar
Data dalam bentuk visual seperti foto, poster, dan grafik.
Contoh: Poster kegiatan sekolah yang diunduh dari internet.

🎧 Data Audio
Data berupa suara atau rekaman.
Contoh: Rekaman pidato presiden dalam format MP3.

🎥 Data Video
Data berupa gambar bergerak dan suara.
Contoh: Video pembelajaran di YouTube.
B. Berdasarkan Sumbernya

📌 Data Primer
Data yang diperoleh langsung dari sumber utama.
Contoh: Hasil wawancara siswa tentang kebiasaan membaca.

🌐 Data Sekunder
Data yang diperoleh dari sumber tidak langsung seperti internet atau buku.
Contoh: Data jumlah penduduk dari website BPS.

2️⃣ Mengakses Data dari Internet

📥 Pengertian

Mengakses data dari internet adalah kegiatan mencari, membuka, dan mengambil informasi dari website atau sumber online yang terpercaya untuk digunakan sesuai kebutuhan.
🔍 Cara Mengakses Data

✔ Menggunakan mesin pencari (Google, Bing).
✔ Mengunjungi website resmi (sekolah, pemerintah, lembaga).
✔ Mengunduh file data (PDF, Excel, CSV).
🧪 Contoh

Seorang siswa ingin mengetahui jumlah penduduk Indonesia per provinsi.
➝ Siswa membuka Google.
➝ Mengetik: “Jumlah penduduk Indonesia BPS”.
➝ Mengunjungi website bps.go.id.
➝ Mengunduh tabel data dalam bentuk Excel.
⚠️ Etika dan Keamanan

✔ Periksa kebenaran sumber data.
✔ Jangan menyebarkan data pribadi.
✔ Cantumkan sumber saat menggunakan data orang lain.

3️⃣ Mengelola Data

📥 Pengertian

Mengelola data adalah kegiatan mengatur, mengolah, menyimpan, dan menyajikan data agar mudah dipahami dan digunakan.
📊 Langkah-Langkah Mengelola Data

1️⃣ Mengumpulkan data → Mengambil data dari survei atau internet.
2️⃣ Mengelompokkan data → Mengatur data berdasarkan kategori.
3️⃣ Menyimpan data → Menyimpan di folder, Excel, Google Sheets.
4️⃣ Menyajikan data → Membuat tabel, grafik, laporan.
🧪 Contoh

Data hasil survei kebiasaan membaca siswa diketik ke Excel, lalu dibuat grafik batang untuk mengetahui kebiasaan membaca terbanyak di kelas.
